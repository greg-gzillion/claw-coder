# 🦞 Claw-Coder

## Intelligent Assistant for PhoenixPME on TX Blockchain

Claw-Coder is an autonomous development agent that analyzes, validates, and optimizes the PhoenixPME auction system on the TX blockchain.

---

## 📋 Features

| Agent | Function | Status |
|-------|----------|--------|
| Auto-Fix Agent | Scans 166+ Rust contracts for bugs | ✅ Active |
| Test Generator | Creates integration tests | ✅ Active |
| Governance Monitor | Tracks CRF & PHNX distribution | ✅ Active |
| Optimization Agent | Finds gas inefficiencies (15-30% savings) | ✅ Active |
| Deep Analysis | Compares docs vs implementation | ✅ Active |
| Multi-Currency Gateway | BTC/ETH/XRP → TESTUSD | ✅ Active |

---

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/greg-gzillion/claw-coder.git
cd claw-coder
chmod +x *.sh

# Run all agents
./run_all_agents.sh
```

---

## 📊 Commands

```bash
# System Management
./run_all_agents.sh     # Run all agents
./status.sh              # Check system status
./validate_contracts.sh  # Validate contract logic
./apply_fixes.sh         # Apply auto-fixes

# Individual Agents
python3 governance_monitor.py   # Track CRF & PHNX
python3 optimization_agent.py   # Find gas optimizations
python3 auto_fix_agent.py       # Find bugs
python3 test_generator.py       # Generate tests
```
